# Lab 20: Configuring NFS for File Sharing

## Objectives
By the end of this lab, you will be able to:
- Install and configure an NFS server and client on Linux systems.
- Export directories from the NFS server and mount them on a client system.
- Optimize NFS settings for security and performance.

## Prerequisites
- Two Linux systems (server and client) with root/sudo access.
- Basic familiarity with Linux command line.
- Network connectivity between the systems (ping test recommended).
